Play Like a Pro

âœ¨ The players.json file in the Chess Server contains thousands of games by titled FIDE players. This file can be generated and customized with the command line tools available in the Chess Data repo.

Chess\Grandmaster figures out the next move to be made based on the JSON file that is passed to its constructor. Please make sure to first create one for it or feel free to use the players.json linked above.

use Chess\Grandmaster;
use Chess\Variant\Classical\Board;

$board = new Board();
$board->play('w', 'e4');

$move = (new Grandmaster(__DIR__.'/../data/players.json'))->move($board);

print_r($move);
